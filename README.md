## ðŸš€ Getting Started

### Prerequisites

You need to have Rust and Cargo installed.

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Installation

1.  Clone the repo:

    ```bash
    git clone https://github.com/matthewskfox/rust-algorithms.git
    cd rust-algorithms
    ```

2.  Run the tests to ensure everything is working:

    ```bash
    cargo test
    ```

3.  To run a specific solution:

    ```bash
    cargo run --bin platform_name
    ```
